# Giat Muda Entrepreneur (GME) Website

Website profil perusahaan untuk Giat Muda Entrepreneur, sebuah platform solusi bisnis digital untuk UMKM.

## Teknologi
- **HTML5**: Struktur halaman.
- **Tailwind CSS**: Framework CSS untuk styling (via CDN).
- **JavaScript (Vanilla)**: Interaktivitas (Menu Mobile, Smooth Scroll, Form Alert).

## Struktur File
- `index.html`: Halaman utama.
- `css/js/script.js`: Logika JavaScript.
- `css/style.css`: Stylesheet tambahan (opsional/legacy).

## Cara Menjalankan (Lokal)
1.  Clone atau download repository ini.
2.  Buka `index.html` langsung di browser Anda.
3.  Atau gunakan ekstensi "Live Server" di VS Code untuk pengalaman yang lebih baik.

## Deployment ke Vercel
Proyek ini siap untuk dideploy ke [Vercel](https://vercel.com).
1.  Install Vercel CLI: `npm i -g vercel`
2.  Jalankan perintah `vercel` di terminal root folder ini.
3.  Ikuti instruksi di layar.
